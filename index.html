CSS Fonts & Text References

Fonts Settings

Property Example of value Description
font-family Arial, Helvetica, sans-serif Specifies the font family for text
font-size 16px, 1.2em, large Sets the size of the font
font-weight normal, bold, 400, 700 Sets how thick or thin characters should be
font-style normal, italic, oblique Specifies the font style for text

Spacing Settings

Property Example of value Description
letter-spacing normal, 0.2em, 2px, -3px Define the space within the letters
word-spacing normal, 0.4em, 3px, -2px Indicate the space between the words
